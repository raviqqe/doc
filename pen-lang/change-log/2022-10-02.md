# String concatenation

- Pen now has a string concatenation operator (`+`.)
- Multiple string concatenations are batched into one.

```pen
concatenateStrings = \(ss [string]) string {
  if [s, ...ss] = ss {
    s() + concatenateStrings(ss)
  } else {
    ""
  }
}
```
